📖 Overview

This repository contains my personal portfolio website built to showcase my work as a Senior Data Engineer specializing in AI platforms, Databricks Lakehouse architectures, and modern data engineering systems.

The website automatically fetches content from multiple platforms including:

GitHub Projects

Medium Articles

YouTube Videos

Documentation repositories

The goal of this portfolio is to act as a living technical profile where projects, articles, and research automatically appear without manual updates.

🧠 Key Features

✔ Auto-sync GitHub repositories into project section
✔ Automatically display Medium articles
✔ Fetch latest YouTube videos
✔ Modern developer portfolio design
✔ Project documentation pages
✔ Blog system using markdown
✔ Fully responsive design
✔ Optimized for SEO

🚀 How to Run Locally

Clone the repository:

git clone https://github.com/harshitboots/harshit-portfolio.git
cd harshit-portfolio

Install dependencies:

npm install

Run development server:

npm run dev

Open in browser:

http://localhost:3000
